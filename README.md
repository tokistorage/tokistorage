# TokiStorage

**存在証明の民主化** — Democratizing Proof of Existence

声・画像・テキストを、物理・国家・デジタルの三層で1000年保管する。

## Repositories

| Repo | Description | Live |
|------|-------------|------|
| [lp](https://github.com/tokistorage/lp) | Landing page, 114+ essays, use cases | [tokistorage.github.io/lp](https://tokistorage.github.io/lp/) |
| [qr](https://github.com/tokistorage/qr) | TokiQR — Voice/image/text to QR code (Codec2 WASM, offline) | [tokistorage.github.io/qr](https://tokistorage.github.io/qr/) |

## Three-Layer Storage

| Layer | Method | Durability |
|-------|--------|-----------|
| Physical | Quartz glass / UV-laminated QR stickers | 1000+ years |
| National | National Diet Library of Japan (NDL) | As long as the institution endures |
| Digital | GitHub Pages | Distributed & open |
